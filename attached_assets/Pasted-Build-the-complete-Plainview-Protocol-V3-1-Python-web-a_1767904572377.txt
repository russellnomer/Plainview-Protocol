PART 2: THE WATCHDOG (`watchdog.py`)
Write a script that:
- Loads `sources.json`.
- Pings each URL using `requests.get()`.
- Creates/Updates a `system_status.json` file: {"treasury": "OK", "senate": "OK", "wiki": "OK"} (or "ERROR").
- If a URL fails, print a warning to the console.

PART 3: THE MAIN APP (`app.py`)
- Import `streamlit`, `pandas`, `requests`, `json`, `plotly.express`.
- Use `@st.cache_data(ttl=3600)` for all data fetching functions to prevent rate limiting.

DATA FETCHING LOGIC (Robust & Keyless):
- `get_live_debt()`: Fetch from `sources.json['treasury_debt']`. Return the "tot_pub_debt_out_amt".
  - *Fallback:* If fetch fails, return hardcoded 36500000000000.00.
- `get_senate_votes()`: Fetch XML from `sources.json['senate_feed']`. Parse top 3 bill titles.
  - *Fallback:* Return list ["Vote data currently unavailable - Check back later"].
- `get_reps(state)`: Use `pd.read_html` on `sources.json['wiki_reps']`. Filter by State.
  - *Fallback:* Return a generic DataFrame: ["Rep. Doe", "District 1", "N/A"].
